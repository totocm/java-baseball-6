-- AllForOne.java
문자열을 readLine() 메서드를 사용하여 받아오고
랜덤 값인 List와 문자열을 비교할 수 있도록 형변환을 시켜줬고
(length는 1부터 시작이고 index는 0부터 시작이라)
인덱스로 비교하며 값들을 더하고 최종에 출력하였다.
테스트를 하며 예외가 터지는 부분을 직접 예외처리 해줬으며
다양한 예외의 상황에 대해 생각해보며 추가하였다.
